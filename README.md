# Project Overview

This project aims to predict used car prices using Machine Learning techniques.
The dataset was collected, cleaned, explored, and preprocessed before training a regression model.

The main objective is to build a reliable Multi-Linear Regression model capable of estimating car prices based on features such as mileage, brand, and other attributes.


---
